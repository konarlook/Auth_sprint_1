## Запуск проекта

1. Рабочая директория `.../Auth_sprint_1`
2. Создать `.env` по `.env.example`
3. Выполнить команду `docker compose up -d --build`

## Запуск тестов Auth

1. Рабочая директория `.../auth/tests/functional`
2. Создать `.env` по `.env.example`
3. Выполнить команду `docker compose up -d --build`
